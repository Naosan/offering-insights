Terms of Service

These terms apply to Offering Insights, a prototype video research dashboard. By using the website or prototype, you agree to these terms.

Offering Insights is an early prototype for reviewing selected public YouTube sources and writing user-authored research notes. It is not affiliated with YouTube or Google.

Prototype status

Offering Insights is provided as a prototype and may change, pause, or stop without notice. Public metadata shown in the workspace is produced from fresh YouTube Data API requests made during the current page session.

The service keeps its project API credential in managed server configuration. Visitors do not enter, receive, or inspect that credential. The current prototype does not provide Google OAuth sign-in, video upload, account management, or comment moderation features.

API Project and data handling

Offering Insights uses Google Cloud project number 141682939002 for the YouTube Data API requests made by this API Client. Offering Insights does not use another project number for this API Client or distribute its YouTube API use across multiple projects. Each metadata lookup makes fresh read-only requests, and the service does not persist selected source IDs or returned API Data in an application database or server-side cache. Details are provided in the Privacy Policy.

Users may save and reopen a local project file containing selected public source IDs, research settings, and user-authored notes. The project file remains on the user's device, is not uploaded by Offering Insights, and does not contain returned YouTube API Data. Reopening a project requires a new, user-initiated metadata lookup before current public YouTube details are displayed.

Acceptable use

You may use the prototype for lawful source review, editorial planning, research notes, and product evaluation. You may not use it to violate YouTube policies, bypass access controls, scrape YouTube outside approved API methods, harass users, or process data in a way that infringes rights or privacy.

Video titles, channel details, category labels, thumbnails, and public statistics displayed as source evidence are YouTube API Data. Offering Insights displays that metadata one source at a time. It does not aggregate category data, calculate source-set representation, rank or score sources, identify trends, or generate candidate angles or recommendations from YouTube API Data. Research topics, questions, per-source notes, follow-up checks, and conclusions are entered by the user and are not YouTube-provided conclusions or endorsements.
