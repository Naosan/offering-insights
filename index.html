What it does

Turn selected videos into a useful planning brief

Offering Insights is built for people preparing editorial planning, source review, or research notes. The workspace keeps the original YouTube sources visible while turning public metadata into a compact direction.

01

Select public videos

Choose public video URLs, video IDs, or a public playlist URL outside the app and paste them into the dashboard.

02

Compare context

Read title, channel, playlist, category, publication, and public statistics context.

03

Prepare the brief

Use the structured view and source map to prepare editorial planning notes or a short evidence brief.

Source evidence

The workspace reads public metadata only

The first version is designed to read public metadata. It does not upload videos, edit channels, moderate comments, or access private account data.

youtube.videos.list

Retrieve public video metadata, statistics, category IDs, and publication details.

youtube.channels.list

Review public channel metadata needed for source context and editorial comparison.

youtube.playlistItems.list

Read public playlist items only when the user provides a public playlist URL as source-set input.

youtube.videoCategories.list

Map category IDs to readable category names for category-level analysis.

Data boundary

Designed for limited, understandable use

The prototype uses approved API methods through a managed service, avoids scraping, and keeps account access out of the first version.

  • Uses YouTube Data API Services endpoints instead of scraping.
  • Keeps the project API credential in managed server configuration, outside the visitor's browser.
  • Does not require users to sign in with Google in the initial version.
  • Provides a public privacy policy, terms page, and contact email.
