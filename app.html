Offering Insights

Public video source review workspace

Turn selected public YouTube videos into source-backed planning briefs

Paste public videos or a public playlist you already selected. Offering Insights reads public metadata, category labels, channel context, thumbnails, and source links, then organizes the set into evidence tables and an exportable brief.

Designed for selected source sets

Workflow
1. Collect Start with a small set of selected public videos.

Use YouTube IDs, URLs, or a public playlist from videos you are already considering for an article, report, or planning note.

2. Understand See which categories and channels shape the set.

The workspace compares public metadata so the team can see whether the set reads as education, science, talks, or a mixed brief.

3. Create Turn the evidence into a useful next step.

Export a concise brief, inspect cited video cards, and use the interactive map to explore candidate source angles.
