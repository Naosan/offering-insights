Offering Insights

Live API-powered workbench

Turn a YouTube source list into a category brief

For operators who already have public YouTube videos and need a fast, evidence-backed category readout they can export.

Data boundary Public metadata only

What the service does

No OAuth or search.list
1. Source set Start with videos the operator already selected.

The service does not discover videos by keyword. It accepts known public YouTube IDs or URLs as the review source set.

2. API evidence Use read-only metadata to explain the source set.

The browser calls video, category, channel, and playlist item endpoints, then records the endpoint trail beside the results.

3. Deliverable Return a category brief that can be exported.

The output summarizes category concentration, public channel context, adjacent upload context, and a concise observation note.
