Privacy Policy

Offering Insights is a prototype video research dashboard. This policy explains how the service handles information in connection with its planned use of YouTube Data API Services.

Current prototype scope: public metadata lookup only, no Google OAuth sign-in, no private YouTube account data, and no upload or account-management functions.

Information we collect

The temporary public website does not require account registration, does not use advertising cookies, and does not ask visitors to sign in with Google. If you contact us by email, we receive the information you choose to send, such as your email address and message.

YouTube Data API Services

The API client is intended to use YouTube Data API Services to retrieve public metadata such as video titles, video IDs, channel IDs, category information, playlist item metadata, publication dates, and public statistics. The initial version does not request Google OAuth access and does not access private YouTube account data.

Visitors provide public YouTube video IDs, video URLs, or public playlist URLs for analysis. The static review build includes a separate developer setup in which the project owner may activate one existing project API key for the current page session. The application does not generate an API key for each visitor or for each analysis request. The key is held only in page memory, is sent directly to Google API endpoints, and is not persisted by this static website or transmitted to an Offering Insights server.

Use of YouTube Data API Services is also governed by the YouTube Terms of Service, the YouTube API Services Terms of Service, and the Google Privacy Policy.

How we use information

Public YouTube metadata is used to understand category-level patterns, compare public channel context, and prepare editorial, category, or research notes. We do not sell personal information. We do not use Google user data for advertising targeting. Human access to operational data is limited to maintenance, security, policy, or user support needs.

Storage and deletion

The static demo shown on this website stores no YouTube API response data. If caching is added to the prototype, cached public API response data will be limited to what is needed for product functionality, security, and policy obligations. Requests for deletion or correction can be sent to offeringinsights.contact@gmail.com.

Data sharing

We do not transfer Google user data to third parties except as required for operating the service, complying with law, security review, or with the user's explicit direction. The initial prototype does not collect Google OAuth user data.

Contact

For privacy questions, data deletion requests, or policy questions, contact offeringinsights.contact@gmail.com.
