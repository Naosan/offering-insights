Privacy Policy

Offering Insights is a prototype video research dashboard. This policy explains how the service handles information in connection with its planned use of YouTube Data API Services.

Current prototype scope: public metadata lookup only, no Google OAuth sign-in, no private YouTube account data, and no upload or account-management functions.

Information we collect

The temporary public website does not require account registration, does not use advertising cookies, and does not ask visitors to sign in with Google. If you contact us by email, we receive the information you choose to send, such as your email address and message.

YouTube Data API Services

The API client uses YouTube Data API Services to retrieve public metadata such as video titles, video IDs, channel IDs, category information, playlist item metadata, publication dates, and public statistics. The current version does not request Google OAuth access and does not access private YouTube account data.

Visitors provide public YouTube video IDs, video URLs, or public playlist URLs for analysis. The static review build includes a separate developer setup in which the project owner may activate one existing project API key before a visitor-flow demonstration in the current page session. The application does not generate an API key for each visitor or for each analysis request. The key is held only in page memory, is transmitted directly to Google API endpoints over HTTPS in a request header rather than the request URL, and is not persisted by this static website or transmitted to an Offering Insights server. A production public release would keep the project credential in a managed server component so visitors never enter, receive, or inspect it.

Use of YouTube Data API Services is also governed by the YouTube Terms of Service, the YouTube API Services Terms of Service, and the Google Privacy Policy.

How we use information

The public source IDs parsed from visitor-supplied URLs, the selected category region, and the required read-only request parameters are sent directly to Google API endpoints. Returned public YouTube metadata is used to organize the selected source set, compare public channel context, and prepare editorial, category, or research notes. We do not sell personal information. We do not use Google user data for advertising targeting. Human access to operational data is limited to maintenance, security, policy, or user support needs.

Storage and deletion

The static demo does not persist YouTube API response data. Live results remain in the current page session, and a report is saved locally only when the visitor chooses the copy or download action. If caching is added later, cached public API data will be limited to what is needed for product functionality and will be refreshed or deleted within the applicable YouTube policy period. Requests for deletion or correction can be sent to offeringinsights.contact@gmail.com.

Google account permissions

This version does not request Google OAuth access or Authorized Data. Google account permissions can be reviewed or revoked at any time from Google's security settings page.

Data sharing

Public source IDs and the request parameters needed for the selected analysis are transmitted to Google through YouTube Data API Services. We do not transfer Google user data to other third parties except as required for operating the service, complying with law, security review, or with the user's explicit direction. The current prototype does not collect Google OAuth user data.

Contact

For privacy questions, data deletion requests, or policy questions, contact offeringinsights.contact@gmail.com.
